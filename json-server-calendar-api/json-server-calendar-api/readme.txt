HOW TO START IT:

- run in terminal "npm run server"
- go to http://localhost:5000/

ABOUT:

- file "db.json" is our "fake database"
- to access users go to http://localhost:5000/users
- you can access a specific user by id (ex. http://localhost:5000/users/3)
- for more examples import the postman file into Postman App
